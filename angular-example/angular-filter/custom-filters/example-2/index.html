AngularJS Filter - Capitalize

Original:

  • {{example1}}
  • {{example2}}

Capitalize Filter:

  • {{example1| capitalize}} - Default - Capitalizes the first letter
  • {{example1| capitalize:5}} - Capitalizes the 5th letter
  • {{example1| capitalize:14}} - Out of bounds, nothing gets capitalized
  • {{example2| capitalize}} - Default - If first, letter is capital, does nothing
  • {{example2| capitalize:17}} - Capitalizes the 17th letter
